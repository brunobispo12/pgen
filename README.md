# Pgen

## How to run

<ul>
  <li>
    Make sure rust is installed in your computer
  </li>
  <li>
    Run: 
    
  ```
    cargo build --release
  ```
  </li>
  <li>
    Add the pgen.exe to PATH
  </li>
  <li>
    Run:
    
      pgen
  </li>
</ul>

## Commands

<ul>
  <li>
    <strong>--length:<strong/> Define the length of the password, short: -l
  </li>
  <br/>
  <li>
    <strong>--symbols:<strong/> Add symbols to the generated password, short: -s
  </li>
</ul>
